5. 数据采集系统

系统要求

系统概述

1

数据源配置

执行环境： VS Code编辑器和终端

前提条件： 已激活Python虚拟环境(venv)

1. 安装依赖：

pip install requests aiohttp pydantic

2. 配置数据源：

# 在VS Code中创建config.py from dataclasses import dataclass from typing import Optional @dataclass class APIConfig: name: str base_url: str api_key: str rate_limit: int
2

爬虫系统

执行环境： VS Code终端或Windows命令提示符(CMD)

前提条件： 已安装Scrapy和Redis

1. 安装爬虫依赖：

pip install scrapy redis scrapy-redis

2. 创建Scrapy项目：

# 在项目根目录执行 scrapy startproject sports_crawler cd sports_crawler scrapy genspider match_spider example.com
3

存储系统

执行环境： Windows PowerShell（管理员）

前提条件： 已安装MongoDB和Redis

1. 启动MongoDB服务：

net start MongoDB

2. 启动Redis服务：

net start Redis
