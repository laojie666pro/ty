3.1 开发工具安装

LMStudio 安装配置

1. 下载安装

  1. 访问 LMStudio官网
  2. 下载最新版本
  3. 运行安装程序：
    • 选择安装路径（建议安装在非系统盘）
    • 创建桌面快捷方式
    • 添加到开始菜单

2. 初始配置

  1. 首次启动设置：
    • 选择中文界面
    • 设置模型存储路径
    • 配置GPU加速
  2. 性能设置：
    • 最大内存使用：28GB（系统内存的70%）
    • GPU内存限制：6GB
    • 线程数：6

3. API配置

设置API访问参数：

端口: 8080 主机: localhost 并发数: 5

验证安装

  1. 检查GPU是否被正确识别
  2. 测试模型加载功能
  3. 验证API接口可访问性

Visual Studio Code 安装（可选）

1. 安装VSCode

  1. 访问 VSCode官网
  2. 下载Windows版本
  3. 运行安装程序，选择：
    • ✓ 添加到PATH
    • ✓ 添加到右键菜单
    • ✓ 注册为默认编辑器

2. 安装扩展

  • Python
  • Jupyter
  • Python Indent
  • GitLens
  • Chinese Language Pack

常见问题

LMStudio无法识别GPU

检查以下内容：

  1. NVIDIA驱动是否最新
  2. CUDA是否正确安装
  3. 是否有其他程序占用GPU
内存分配错误

调整以下设置：

  • 降低最大内存使用限制
  • 减少并发数
  • 使用更小的模型
