3. 开发环境配置

配置前准备

配置步骤

1. Python开发环境

执行环境： VS Code的终端或Windows命令提示符(CMD)

前提条件： 已激活Python虚拟环境(venv)

安装开发依赖：

# 确保在项目根目录下执行 pip install -r requirements-dev.txt

2. IDE配置

执行环境： VS Code编辑器

操作步骤： 按Ctrl+Shift+P打开命令面板，输入"settings.json"

将以下配置复制到VS Code的settings.json文件中：

{ "python.defaultInterpreterPath": "./venv/bin/python", "python.linting.enabled": true, "python.linting.pylintEnabled": true, "python.formatting.provider": "black", "editor.formatOnSave": true, "editor.rulers": [88], "files.trimTrailingWhitespace": true }

3. Git配置

执行环境： VS Code的终端或Windows命令提示符(CMD)

1. 配置用户信息：

git config --global user.name "Your Name" git config --global user.email "your.email@example.com"

2. 配置行尾处理（Windows系统必须）：

git config --global core.autocrlf input

3. 在项目根目录创建.gitignore文件：

# 使用VS Code或记事本创建并添加以下内容： venv/ __pycache__/ *.pyc .env .vscode/ .idea/ *.log dist/ build/ *.egg-info/

4. 预提交钩子配置

执行环境： VS Code的终端或Windows命令提示符(CMD)

前提条件： 已激活Python虚拟环境(venv)

1. 安装pre-commit：

pip install pre-commit

2. 在项目根目录创建.pre-commit-config.yaml文件：

# 使用VS Code或记事本创建并添加以下内容： repos: - repo: https://github.com/psf/black rev: 22.3.0 hooks: - id: black language_version: python3.8 - repo: https://github.com/pycqa/isort rev: 5.10.1 hooks: - id: isort - repo: https://github.com/pycqa/flake8 rev: 4.0.1 hooks: - id: flake8 additional_dependencies: [flake8-docstrings]

3. 安装git hooks：

pre-commit install
