2. 基础环境安装

1. Python安装

执行环境： Windows系统下的图形界面安装

  1. 访问 Python官网
  2. 下载Python 3.8或更高版本的Windows安装包
  3. 运行安装包，勾选"Add Python to PATH"
  4. 点击"Install Now"完成安装

验证安装：

在Windows命令提示符(CMD)中执行：

python --version pip --version

2. CUDA安装

执行环境： Windows系统下的图形界面安装

  1. 访问 NVIDIA CUDA下载页面
  2. 选择对应的Windows版本下载安装包
  3. 运行安装包，按提示完成安装

验证安装：

在Windows命令提示符(CMD)中执行：

nvcc --version nvidia-smi

3. 项目依赖安装

执行环境： VS Code的终端或Windows命令提示符(CMD)

注意： 建议在项目目录下创建虚拟环境后再安装

1. 创建虚拟环境（在项目目录下执行）：

python -m venv venv

2. 激活虚拟环境：

# Windows CMD中： venv\Scripts\activate # Windows PowerShell中： .\venv\Scripts\Activate.ps1

3. 安装依赖：

pip install -r requirements.txt

4. Git安装

执行环境： Windows系统下的图形界面安装

  1. 访问 Git官网
  2. 下载Windows版本的安装包
  3. 运行安装包，使用默认选项安装

验证安装：

在Windows命令提示符(CMD)或VS Code终端中执行：

git --version
