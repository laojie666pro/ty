2.3 Python环境安装

安装前检查

安装步骤

1. 安装Python 3.9.13

  1. 访问 Python 3.9.13下载页面
  2. 下载 Windows installer (64-bit)
  3. 运行安装程序，选择以下选项：
    • ✓ Add Python 3.9 to PATH
    • ✓ Install launcher for all users
    • 选择"Customize installation"
  4. 在高级选项中确保选中：
    • ✓ pip
    • ✓ py launcher
    • ✓ Install for all users
    • ✓ Create shortcuts for installed applications
    • ✓ Add Python to environment variables

2. 配置虚拟环境

  1. 打开命令提示符
  2. 创建项目目录：
    mkdir D:\sports_prediction cd D:\sports_prediction
  3. 创建虚拟环境：
    python -m venv sports_pred_env
  4. 激活虚拟环境：
    sports_pred_env\Scripts\activate

3. 安装基础依赖

pip install --upgrade pip pip install numpy==1.24.3 pip install pandas==2.0.3 pip install scikit-learn==1.3.0

4. 安装PyTorch

pip install torch==2.0.1 torchvision==0.15.2 torchaudio==2.0.2 --index-url https://download.pytorch.org/whl/cu117
注意： 确保使用CUDA 11.7版本对应的PyTorch包

5. 安装其他依赖

pip install transformers==4.33.2 pip install datasets==2.14.5 pip install pymongo==4.5.0 pip install requests==2.31.0 pip install beautifulsoup4==4.12.2 pip install selenium==4.12.0 pip install fastapi==0.103.1 pip install uvicorn==0.23.2

验证安装

1. 检查Python环境

python --version pip list

2. 验证PyTorch CUDA支持

python -c "import torch; print(f'CUDA是否可用：{torch.cuda.is_available()}')"

常见问题

pip安装包失败

可能原因：

  • 网络问题
  • 版本冲突
  • Python版本不匹配

解决方案：

  1. 使用国内镜像源
  2. 检查包版本兼容性
  3. 确认Python版本要求
CUDA不可用

检查以下内容：

  1. NVIDIA驱动是否正确安装
  2. CUDA版本是否与PyTorch匹配
  3. 是否使用了正确的PyTorch安装命令
