2.2 CUDA安装

安装前检查

安装步骤

1. 下载CUDA 11.7

  1. 访问 CUDA 11.7下载页面
  2. 选择下载选项：
    • 操作系统：Windows
    • 架构：x86_64
    • 版本：11.7.0
    • 安装类型：exe (local)
  3. 下载安装程序（约3GB）

2. 安装CUDA

  1. 以管理员身份运行安装程序
  2. 选择安装类型：自定义安装
  3. 确保以下组件被选中：
    • CUDA Runtime and Development
    • Visual Studio Integration
    • CUDA Samples
  4. 选择安装路径（建议默认路径）
  5. 等待安装完成

3. 安装cuDNN

  1. 访问 cuDNN下载页面（需要注册NVIDIA账号）
  2. 下载对应CUDA 11.7的cuDNN版本
  3. 解压下载的文件
  4. 复制文件到CUDA目录：
    将解压后的 cuda 文件夹中的内容复制到 C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.7

4. 环境变量配置

确保以下路径已添加到系统环境变量 Path：

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.7\bin C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.7\libnvvp C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.7\include

验证安装

1. 检查CUDA版本

nvcc --version

2. 编译CUDA示例

cd "C:\ProgramData\NVIDIA Corporation\CUDA Samples\v11.7" make

3. 运行带宽测试

cd bin\win64\release bandwidthTest.exe

常见问题

nvcc命令未找到

检查环境变量是否正确配置：

  1. 打开系统环境变量设置
  2. 检查Path变量中的CUDA路径
  3. 重新启动命令提示符
CUDA示例编译失败

可能原因：

  • Visual Studio未正确安装
  • 缺少Windows SDK
  • PATH环境变量配置错误
