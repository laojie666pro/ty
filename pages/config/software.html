1.2 软件要求

操作系统

项目 要求 说明
系统版本 Windows 10/11 专业版或企业版
系统位数 64位 必需
更新版本 最新 建议开启自动更新

驱动程序

组件 版本要求 下载地址
NVIDIA驱动 最新版本 官方下载
CUDA Toolkit 11.7 官方下载
cuDNN 8.9.x 官方下载

开发环境

软件 版本 说明
Python 3.9.x 建议使用3.9.13
Git 最新版本 用于代码管理
Visual Studio Build Tools 2019或更高 编译依赖项必需

安装顺序

  1. 更新Windows系统
  2. 安装NVIDIA驱动
  3. 安装CUDA Toolkit
  4. 安装cuDNN
  5. 安装Python环境
  6. 安装其他开发工具
提示： 建议在安装前关闭杀毒软件，以避免安装过程中的干扰。
